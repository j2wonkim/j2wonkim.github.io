svn에서 commit할 때 순서가 꼬인 상태에서 확인하여 해결하는 방법

1. remote에서 업데이트 받은 로컬의 상태 먼저 확인           
```
svn status --show-updates
```

2. 현재 디렉토리(프로젝트 root 경로) clean up           
```
svn cleanup .
```
